Casa

Esta imagen es el resultado de una actividad diseñada para aprender a utilizar SVG (Scalable Vector Graphics) en la web. SVG es un formato basado en XML que permite crear gráficos vectoriales directamente en el código, sin necesidad de imágenes externas. En este ejercicio, se programaron formas básicas para construir una casa:
  • Rectángulos para fondo, las paredes, las ventanas y puerta.
  • Un triángulo para el techo.
  • Círculos para representar el sol y el pomo de la puerta.
    • El objetivo de esta actividad es comprender cómo se pueden dibujar y manipular figuras con SVG usando código, permitiendo gráficos escalables y optimizados para la web.

    Esta imagen es parte de una actividad diseñada para aprender a programar gráficos con SVG (Scalable Vector Graphics). SVG permite dibujar formas mediante código, sin necesidad de imágenes externas, lo que lo hace ideal para gráficos escalables y optimizados para la web. En este ejercicio, se utilizaron diferentes elementos para construir un coche:
  • Rectángulos rojos para la carrocería y la cabina.
  • Rectángulos azules para simular las ventanas.
  • Rectángulos amarillos como faros delanteros y traseros.
  • Círculos grises y negros para representar las ruedas con su centro.
    • Con esta práctica, se explora cómo combinar formas geométricas simples para crear ilustraciones más complejas, desarrollando habilidades en el uso de SVG y programación gráfica. 🚀

    🚗 Creando un Coche con SVG
    Esta imagen fue creada como parte de una actividad para aprender a programar gráficos con SVG (Scalable Vector Graphics). SVG permite dibujar figuras utilizando código, lo que facilita la creación de gráficos escalables sin pérdida de calidad. En este ejercicio, se construyó un coche usando diferentes formas geométricas:
  • Rectángulos para el chasis, las ventanas y otras partes de la estructura.
  • Círculos para representar luces, botones e indicadores.
    • Con esta actividad, se explora cómo combinar elementos básicos para diseñar objetos más complejos, fomentando el aprendizaje de gráficos vectoriales y programación visual. 🚀
